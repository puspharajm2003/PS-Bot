Introduction

The PS - BOT is a robot designed to assist people for their daily life. this is a cutting-edge personal assistance that combines advanced techology with Voice assistance.The features like AMR processed Robot, Object Detection, Object Tracking,Face Detection and ROS for automatic path finding.You can Communicate with the PS Bot by Voice assistance. The bot is designed to be user friendly and easy.
